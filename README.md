# Damage Calculator

A damage calculator for competitive Pokemon battles.

**Website**: [damage-calc.com](https://damage-calc.com)

## Features

- Offensive power / damage calculation (official Gen V+ formula)
- Speed comparison
- Terastal / Dynamax / Mega Evolution support
- Legends Z-A new Mega Evolutions included
- Korean consonant search
- Responsive UI for mobile and desktop

## Platforms

- Web — [damage-calc.com](https://damage-calc.com)
- Android / iOS — Coming soon

## Sprite Credits

Pokémon sprites are streamed directly from the [Pokémon Showdown sprite CDN](https://play.pokemonshowdown.com/sprites/). This project does not host or redistribute any sprite assets — the app reads them from Showdown's servers at runtime and only on the web build (mobile sprite support is planned and will follow the same no-rehosting model).

The BW (Generation 5 pixel) sprite set used as the default style includes community-contributed sprites from the [Smogon Sprite Project](https://www.smogon.com/forums/threads/smogon-sprite-project.3647722/) and the [X/Y Sprite Project](https://www.smogon.com/forums/threads/x-y-sprite-project.3486712/), which extends BW-style coverage to post-Gen-5 Pokémon. Those community sprites are licensed for non-profit use with credit. The animated (`ani`) and HOME 3D (`dex`) sprite styles are derived works of official Game Freak / The Pokémon Company artwork — same disclaimer below applies.
